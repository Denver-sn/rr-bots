### Installation of the bot to the server
  - Open the console of your server
  - Login as root
  - Type: `crontab -e`
  - Scroll back and add this line: `1 * * * *   python3 /path_of_the_file/file.py`
  - Type `CTRL + X` ==> `Y` ==> and now quit
